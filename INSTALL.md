Building Lekan from source

## Requirement under Linux

#### QGIS

Even is QGIS is already installed, you need to build a proper version of QGIS from this branch: https://github.com/vcloarec/QGIS/tree/forLekan

#### Gmsh

For Lekan, Gmsh version 4.9 is required. If this version is not installed, you need to build it from sources: https://gitlab.onelab.info/gmsh/gmsh
